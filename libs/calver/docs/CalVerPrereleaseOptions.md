# CalVerPrereleaseOptions (extends [VersioningPrereleaseOptions][])

## prefix

**Type**: `"." | "_" | "-" | ""`

**Default**: `"-"`

The separator between the pre-release name and the main version.

## suffix

**Type**: `"." | ""`

**Default**: `""`

The separator between the pre-release name and the pre-release version.
